# CreditSimulationFront

This project is a front end that I decided to create for my [Credit-Simulation-API](https://github.com/LeonardoGarciaMelo/Credit-Simulation-API) and it was created using Angular version 20.3.3.

## Starting Application
First you need to start the [backend](https://github.com/LeonardoGarciaMelo/Credit-Simulation-API/tree/main/Simulador_de_Credito) using

```bash
dotnet run
``` 

Then, to start the front, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/api/simulacoes/simular`.
